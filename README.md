# ❤️ Análise de Risco de Doença Cardíaca Coronariana (DCC) em 10 Anos

Este projeto investiga fatores de risco para doenças cardíacas coronarianas em um período de 10 anos. O objetivo principal é criar um modelo de classificação utilizando Regressão Logística para prever a probabilidade de um paciente desenvolver Doença Cardíaca Coronariana (DCC).

# 📜 Workflow do Projeto

1. Preparação e Limpeza de Dados
2. Separação dos dados em conjuntos de treino e teste 
3. Modelagem de Regressão Logística
   
    3.1 Criação e treinamento de um modelo de Regressão Logística
   
    3.2 Avaliação do Modelo
   
    3.3 Cálculo da acurácia do modelo no conjunto de teste
  
5. Rastreamento do Experimento
6. Predição de Exemplo

# **Insigths Chave**

O modelo de Regressão Logística treinado com todas as features (características dos pacientes) alcançou uma acurácia de aproximadamente **84.79%** na previsão de DCC em 10 anos.
A predição para um paciente de exemplo indicou que ele não desenvolveria DCC.

# Resultados do Projeto:

Desenvolvimento de um modelo de Regressão Logística que considera todas as características do paciente e tem uma acurácia no conjunto de teste de 84.79%

# Conclusão

O modelo de Regressão Logística inicial desenvolvido neste notebook demonstrou uma boa acurácia na previsão de Doença Cardíaca Coronariana. O rastreamento com MLflow facilita a gestão e comparação de futuros experimentos para aprimorar o modelo.
